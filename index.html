المزايا

لماذا GT-SALAT؟

تطوير كامل للتطبيق الأصلي مع الحفاظ على جميع الميزات وإضافة واجهة رسومية تفاعلية حديثة

مواقيت الصلاة

حساب مواقيت الصلاة الخمس بـ 22 طريقة حسابية تشمل أم القرى وISNA ورابطة العالم الإسلامي والمغرب والجزائر وغيرها، مع مصدر مزدوج: API أولاً وحساب محلي احتياطي يعمل دون إنترنت.

الإشعارات والأذان

إشعارات قبل الصلاة بفترة قابلة للضبط، تشغيل الأذان عند دخول الوقت بخيارين (كامل وقصير)، دعم رفع أذان مخصص بصيغ OGG وMP3 وWAV، مع دعاء بعد الأذان وأذكار بعد الصلاة.

الأذكار

قاعدة بيانات كاملة للأذكار مع بحث فوري في النصوص، وعداد لكل ذكر محفوظ محلياً، وعرض ذكر عشوائي في لوحة التحكم الرئيسية.

التكامل مع الطرفية

عند فتح أي طرفية جديدة يظهر تلقائياً: ترويسة «بسم الله الرحمن الرحيم» وذكر عشوائي واسم الصلاة القادمة ووقتها والوقت المتبقي. يدعم bash وzsh وfish.

التكامل مع النظام

أيقونة شريط المهام مع قائمة سياق، تصغير إلى شريط المهام بدلاً من الإغلاق، بدء تشغيل تلقائي مع الجهاز، وفتح مجلد جداول المواقيت مباشرة من الإعدادات.

واجهة عصرية

وضعان: مظلم وفاتح، خطوط Ubuntu Arabic وAmiri Quran مضمّنة تعمل بلا اتصال إنترنت، واجهة عربية RTL كاملة، ومعالج إعداد خطوة بخطوة للتهيئة الأولى.
